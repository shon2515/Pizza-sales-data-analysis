# Pizza-sales-data-analysis
Project Overview

This project analyzes pizza sales data to extract meaningful business insights using SQL and Excel dashboards.
The primary goal is to calculate key performance indicators (KPIs), analyze customer ordering patterns, and identify top-performing pizza categories and products to support data-driven decision-making.

The project focuses on:

KPI calculation using SQL aggregations

Trend analysis by day and hour

Sales distribution by pizza category and size

Identifying best- and worst-selling pizzas

Building an interactive Excel dashboard for business insights.
